^|*Requirement {counter:req-id}* |*/req/landing-page/conformance-success*

^|B |The content of that response SHALL be based upon the schema link:http://beta.schemas.opengis.net/ogcapi/common/part1/0.1/core/openapi/schemas/confClasses.yaml[confClasses.yaml] and list all OGC API conformance classes that the API conforms to.
